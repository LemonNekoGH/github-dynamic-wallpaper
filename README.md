### 可自定义选项
| 选项名称 | 类型 | 默认值 | 注释 |
| --- | --- | --- | --- |
| 时间格式 | 文本 | HH:mm:ss | [格式参考](http://momentjs.cn/docs/#/displaying/) |
| 时间字体大小 | 数字 | 7 | 不要设太大 |
| 时间的X轴偏移量 | 数字 | 26 | 最小值0，最大值100 |
| 时间的Y轴偏移量 | 数字 | 37 | 最小值0，最大值100 |
